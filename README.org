* Synth
[[./demo.png]]

A WIP synthesizer in C.

* Quick start
Depends on [[https://www.libsdl.org/index.php][SDL2]]

#+begin_src console
$ git clone https://github.com/shoumodip/synth
$ cd synth
$ make
$ ./synth
#+end_src

* Usage
Pressing the key ~q~ exits ~synth~.

** White Notes
| Key | Note |
|-----+------|
| ~a~ | C4   |
| ~s~ | D4   |
| ~d~ | E4   |
| ~f~ | F4   |
| ~j~ | G4   |
| ~k~ | A4   |
| ~l~ | B4   |

| Key | Note |
|-----+------|

| ~e~ | E'   |
| ~u~ | F#   |
| ~i~ | A'   |
